테니스 조 추첨

KCS ◆ 모닝스타

전체 조 개수를 정한 뒤 각 조의 이름과 참가 인원을 입력하세요. 원하는 두 조를 같은 페어 그룹으로 묶으면, 각 조를 따로 추첨한 뒤 같은 순번끼리 페어가 됩니다.

1. 조 개수와 조별 인원 설정

최상단에서 전체 조 개수를 정하면 해당 개수만큼 조 설정란이 생성됩니다.

4개 조 · 0명
입력한 수만큼 조 설정란을 자동 생성합니다.

조 이름, 참가 인원수, 선택 사항인 페어 그룹을 설정해 주세요.

🎾 페어 그룹은 선택 사항입니다. 같은 페어 그룹을 선택한 두 조는 인원수가 같아야 하며, 추첨 후 1번끼리, 2번끼리 순서대로 페어가 됩니다. 미지정 조는 조별 순서만 무작위로 추첨합니다.

2. 조별 참가자 이름 입력

각 조에 설정한 인원수만큼 이름칸이 별도로 생성됩니다.

필요한 전체 참가 인원 0명
전체 이름 입력 현황 0 / 0명

각 명단은 조별로 분리됩니다. 페어 그룹을 설정해도 참가자가 다른 조로 이동하지 않고, 무작위 순번만 연결됩니다.

각 조의 참가 인원수를 입력하면 조별 이름 입력칸이 표시됩니다.

3. 추첨 및 페어 생성

각 조의 명단을 무작위로 섞고, 설정된 페어 그룹은 같은 순번끼리 연결합니다.

각 조에 입력한 참가자만 해당 조 안에서 안전한 난수로 섞습니다. 같은 페어 그룹의 두 조는 무작위 결과의 동일 순번끼리 연결하며, 재추첨할 때마다 순서와 페어가 새로 정해집니다.
